# Exercício - Conversão para SASS

## O que foi feito
- ✅ CSS convertido para SASS (arquivos em `./scss`)
- ✅ Variáveis do SASS (`./scss/_variables.scss`)
- ✅ Módulos do SASS com `@use` (arquivo de entrada `./scss/main.scss`)
- ✅ `styles.css` mantido/gerado a partir do SASS para o navegador

## Como compilar o SASS
1. Instale as dependências:
```bash
npm i
```

2. Gerar o CSS:
```bash
npm run build:css
```

3. Modo watch (opcional):
```bash
npm run watch:css
```

> O HTML já aponta para `styles.css`.
